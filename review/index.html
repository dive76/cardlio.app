cardlio — App Review sample files

Sample inputs for reviewing the cardlio scanning pipeline. All cards show fictional people and companies. This location is stable and will remain available for future reviews.

  • sample-cards-multi.jpg — four cards on a table (batch detection)
  • sample-card-1.jpg — single card
  • sample-card-2.jpg — single card
  • sample-card-3.jpg — single card
  • sample-cards.pdf — two-page PDF

Usage: drag any file into the cardlio window (or use Import…). The review screen appears with the extracted fields; Save commits the cards to the library.
